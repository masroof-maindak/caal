# CAAL

This repository comprises select assignments, labs, and projects from my
Computer Architecture & Assembly Language course. The parts I am most proud of
are the userspace multi-tasking scheduler and minimal CPU.

## Dependencies

- `Digital` by Hneeman for the logic design (computer architecture) assignments,
  found [here](https://github.com/hneemann/Digital)
- `DOSbox` for the x86 assembly assignments
